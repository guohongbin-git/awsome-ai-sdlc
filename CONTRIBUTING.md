# 贡献指南

欢迎为Awesome AI SDLC项目做出贡献！

## 如何贡献

1. Fork本仓库
2. 创建你的分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开Pull Request

## 开发环境

- Python 3.8+
- Git

## 代码风格

请遵循PEP 8代码风格指南。

## 测试要求

所有新功能都应包含相应的单元测试。

## 问题报告

请在GitHub Issues中报告问题，并提供详细的复现步骤。
